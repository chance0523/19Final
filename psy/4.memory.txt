OOO, OO, OO이 모두 성공적으로 이루어져야 기억을 할 수 있다.
부호화, 저장, 인출이 모두 성공적으로 이루어져야 기억을 할 수 있다.
부호화 : 정보를 OO 속에 넣음
부호화 : 정보를 기억 속에 넣음
저장 : 정보를 오랜 시간 OO
저장 : 정보를 오랜 시간 보유
인출 : OOO에서 정보를 OO
인출 : 저장소에서 정보를 꺼냄
OOO를 시키지 못하면 OO도 할 수 없고 OO도 할 수 없다.
부호화를 시키지 못하면 저장도 할 수 없고 인출도 할 수 없다.
얼마나 견고하게 OO하느냐에 따라 OO 가능성이 달라짐.
얼마나 견고하게 저장하느냐에 따라 인출 가능성이 달라짐.
OO하는 상황이 어떤 상황이냐에 따라 기억의 성공이 달라짐
인출하는 상황이 어떤 상황이냐에 따라 기억의 성공이 달라짐
기억의 3단계 모형 : OO기억 -> OO기억(OO기억) -> OO기억
기억의 3단계 모형 : 감각기억 -> 단기기억(작업기억) -> 장기기억
작업기억 : ★ working memory . OO적인 측면에 초점을 둠. 작업기억의 차이에서 OO들의 기억차이가 발생. 단기기억 = OO
작업기억 : ★ working memory . 기능적인 측면에 초점을 둠. 작업기억의 차이에서 개인들의 기억차이가 발생. 단기기억 = 용량
단기기억의 용량 제한 : 모든 사람들에게 동일. magic number . 평균 O자리 정도는 딱히 노력을 기울이지 않아도 30초정도는 기억 할 수 있다.
단기기억의 용량 제한 : 모든 사람들에게 동일. magic number . 평균 7자리 정도는 딱히 노력을 기울이지 않아도 30초정도는 기억 할 수 있다.
OO : 기억항목들을 OO성이나 원리에 따라 묶어 재구성. 1980518 -> OOOO OOO
청킹 : 기억항목들을 유사성이나 원리에 따라 묶어 재구성. 1980518 -> 1980 518
OO 오류 : 스포츠 범주에서 (내가 가장 친숙한)수영이 떠오르면 실수 할 수 있음. 내가 외워야하는 항목 외에 OO OO가 회상되고 암기해야하는 항목과 혼선. 기억해야하는 것이 4개인데 5개가 기억남. 하나를 제거하기 어려움.
OO 오류 : 스포츠 범주에서 (내가 가장 친숙한)수영이 떠오르면 실수 할 수 있음. 내가 외워야하는 항목 외에 추가 정보가 회상되고 암기해야하는 항목과 혼선. 기억해야하는 것이 4개인데 5개가 기억남. 하나를 제거하기 어려움.
작업기억의 작용 1 : OO OOO (C~ E~)
작업기억의 작용 1 : 중앙 집행기 (Central executive)
작업기억의 작용 2 : OOOO (P~ L~)
작업기억의 작용 2 : 음운루프 (Phonological loop)
작업기억의 작용 3 : OOOO (E~ B~)
작업기억의 작용 3 : 일화버퍼 (Episodic buffer)
작업기억의 작용 4 : OOO OOO (V~S~ S~)
작업기억의 작용 4 : 시공간 그림판 (Visuo-spatial sketchpad)
OO OO 효과 : 과일 20항목을 주면 처음 한두항목과 마지막 한두항목은 잘 기억나는데 중간에 있는 것들은 잘 기억 안 남
계열 위치 효과 : 과일 20항목을 주면 처음 한두항목과 마지막 한두항목은 잘 기억나는데 중간에 있는 것들은 잘 기억 안 남
처음 들은 항목 - OO 효과. OO 기억에서 인출
처음 들은 항목 - 초두 효과. 장기 기억에서 인출
마지막에 들은 항목 - OO 효과. OO 기억에서 인출
마지막에 들은 항목 - 최신 효과. 단기 기억에서 인출
계열 위치 효과를 OOO OOO OO 이라고도 함.
계열 위치 효과를 기억의 2요인 이론 이라고도 함.
OO 효과는 나중에 테스트하면 사라짐. 기억력에서 가장 큰 효과를 발휘하는 것은 OO 효과이다.
최신 효과는 나중에 테스트하면 사라짐. 기억력에서 가장 큰 효과를 발휘하는 것은 초두 효과이다.
OO 재생 : 들은 OO대로 단어를 말하라, OO 회상 : 가능한 OO 단어를 말하라
계열 재생 : 들은 순서대로 단어를 말하라, 자유 회상 : 가능한 많은 단어를 말하라
계열 재생과 자유 회상에 상관없이 OO OO OO는 OO하게 나타난다.
계열 재생과 자유 회상에 상관없이 계열 위치 효과는 동일하게 나타난다.
철수 VS 영철 : 주어진 정보상으로는 OO한 인물이지만 첫 인상 효과로 인해 지적이고 부지런한 모습이 OO에 제시된 철수가 더 호감
철수 VS 영철 : 주어진 정보상으로는 동일한 인물이지만 첫 인상 효과로 인해 지적이고 부지런한 모습이 처음에 제시된 철수가 더 호감
장기기억의 종류 1 : OO/OO적 기억 - OO적으로 표현 가능한 특정 사실이나 사건과 같은 정보를 OO적으로 재조합 한 것
장기기억의 종류 1 : 외현/명시적 기억 - 언어적으로 표현 가능한 특정 사실이나 사건과 같은 정보를 의식적으로 재조합 한 것
OO 기억 : 학교에서 배우는 것과 같은 OO과 OO에 관한 기억
의미 기억 : 학교에서 배우는 것과 같은 사실과 지식에 관한 기억
OO 기억 : 우리가 했던 OO에 대한 기억
일화 기억 : 우리가 했던 경험에 대한 기억
장기기억의 종류 2 : OO 기억 - 어떤 특별한 OO적 OO이 필요 없는 기억 과정
장기기억의 종류 2 : 암묵 기억 - 어떤 특별한 의식적 노력이 필요 없는 기억 과정
OO 기억 : O을 하는 OO에 대한 기억. 자전거를 타는 법, 신발끈을 묶는 것. 설명하기는 어렵지만 O을 할 수는 있음
절차 기억 : O을 하는 OO에 대한 기억. 자전거를 타는 법, 신발끈을 묶는 것. 설명하기는 어렵지만 O을 할 수는 있음
OO : 이전과 같거나 비슷한 자극에 노출 된 것에 의해 OO이 OO되는 것
점화 : 이전과 같거나 비슷한 자극에 노출 된 것에 의해 회상이 향상되는 것
