구조적, 의문적, 의미적 encoding
구조적 encoding : OO의 OO를 위주로 정보처리
구조적 encoding : 단어의 형태를 위주로 정보처리
의문적 encoding : OO을 위주로 정보처리
의문적 encoding : 음성을 위주로 정보처리
의미적 encoding : OO를 위주로 정보처리
의미적 encoding : 의미를 위주로 정보처리
OO적 encoding이 제일 기억 잘 함.
의미적 encoding이 제일 기억 잘 함.

부호화 특수성 : 기억의 OO효과 - 어떤 사건 회상/재인 시 그 사건이 발생했던 OO과 OO할수록 기억이 잘 되는 현상
부호화 특수성 : 기억의 OO효과 - 어떤 사건 회상/재인 시 그 사건이 발생했던 맥락과 유사할수록 기억이 잘 되는 현상
다이버들에게 해변 혹은 수중에서 단어를 학습하도록 함. 학습했던 OO과 동일한 OO에서 회상 시 더 잘 기억
다이버들에게 해변 혹은 수중에서 단어를 학습하도록 함. 학습했던 맥락과 동일한 맥락에서 회상 시 더 잘 기억
OO곡선 : OOO 철자에 대한 기억 측정. 짧은 시간 내에 급격하게 망각이 이루어짐. 어느정도 망각이 이루어지면 OO적인 곡선
망각곡선 : 무의미 철자에 대한 기억 측정. 짧은 시간 내에 급격하게 망각이 이루어짐. 어느정도 망각이 이루어지면 안정적인 곡선
OO간섭 : OO에 습득한 정보가 OOO 정보를 획득하는 것을 어렵게 만드는 것. A학습 -> B학습 -> O회상 어려움
순행간섭 : 과거에 습득한 정보가 새로운 정보를 획득하는 것을 어렵게 만드는 것. A학습 -> B학습 -> B회상 어려움
OO간섭 : OOO 학습이 OO 학습 내용 회상을 방해. A학습 -> B학습 -> O회상 어려움
역행간섭 : 새로운 학습이 이전 학습 내용 회상을 방해. A학습 -> B학습 -> A회상 어려움
OOOO 망각 : 특정 단어의 OO을 OO하게 되면 그 범주 내에 있는 다른 단어의 OO이 어려운 현상. 특정 챕터의 내용만 공부하면 다른 챕터의 회상량을 떨어뜨린다.
인출유도 망각 : 특정 단어의 인출을 유도하게 되면 그 범주 내에 있는 다른 단어의 인출이 어려운 현상. 특정 챕터의 내용만 공부하면 다른 챕터의 회상량을 떨어뜨린다.
